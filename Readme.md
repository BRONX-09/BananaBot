# Music

use
!play Song name here
this will play the top search result if youtube.

provide the command with a url to play a particular yt audio.

use
!leave to stop the bot and make it leave the voice channel.

if play another song one song is already playing, it will stop the existing song and will play the most recent song.

There are no queues.
